# Единый Rubric Для Отчетов ЛР 01–05

Этот шаблон нужен, чтобы оценивание было прозрачным и одинаковым по всем лабораторным.

## Шкала
- `0` — критерий не выполнен.
- `1` — выполнено частично, много пробелов.
- `2` — базово выполнено, но есть методические/логические пробелы.
- `3` — хорошо выполнено, небольшие недочеты.
- `4` — выполнено полно и корректно.

## Критерии
| Критерий | Что проверяем | Балл (0-4) |
|---|---|---:|
| Полнота артефактов | Все обязательные файлы/таблицы присутствуют, форматы и колонки корректны. |  |
| Корректность интерпретации | Выводы действительно следуют из метрик/таблиц/графиков, нет подмены причинно-следственных связей. |  |
| Обоснование решений | Есть явная логика выбора модели/порога/policy, опора на данные, а не на интуицию. |  |
| Качество рефлексии и ограничений | Указаны риски, ограничения эксперимента и что можно улучшить в следующем цикле. |  |
| Воспроизводимость | Понятно, как были получены результаты; отсутствуют незавершенные TODO/NotImplementedError. |  |

## Итог
- Максимум: `20` баллов.
- Рекомендуемая интерпретация:
  - `17-20` — отлично;
  - `13-16` — хорошо;
  - `9-12` — удовлетворительно;
  - `<9` — требуется доработка.
