# Глоссарий незнакомых вещей (обязательно)

Обновляйте этот файл **по ходу выполнения** ноутбуков, а не в конце.

Минимальные требования:
- добавить не менее 3 новых терминов на каждый ноутбук;
- для каждого термина указать простое объяснение своими словами;
- указать, где термин встретился в работе;
- добавить источник (URL, учебник, статья или ссылка на внутреннюю заметку).

Рекомендуемые термины для старта:
- `overfitting`
- `underfitting`
- `validation set`
- `generalization gap`
- `candidate feature set`
- `GridSearchCV`
- `Pipeline`
- `data leakage`

Подсказка:
если в объяснении термина хочется написать "`Pipeline` автоматически решает leakage",
это слишком сильное упрощение. Правильнее объяснять, какие именно шаги были обучаемыми
и почему важно поместить их внутрь `fit`/CV.
